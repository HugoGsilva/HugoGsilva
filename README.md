# Sobre

Sou estudante de Análise e Desenvolvimento de Sistemas no IFPR Campus Paranaguá.

Tenho foco em desenvolvimento web, back-end e infraestrutura. Atualmente venho trabalhando mais com TypeScript, APIs, Docker, Portainer e deploy de aplicações.

Gosto de montar projetos completos, integrando front-end, back-end, banco de dados e ambiente de execução.

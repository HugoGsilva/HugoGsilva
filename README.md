## Sobre mim

Sou estudante de Análise e Desenvolvimento de Sistemas no IFPR Campus Paranaguá.

Tenho interesse em desenvolvimento web, back-end e infraestrutura. Atualmente venho estudando mais TypeScript, Docker, Docker Compose, Portainer e cloud.

Gosto de aprender fazendo projetos práticos, principalmente quando envolvem APIs, bancos de dados, automação ou deploy de aplicações.

## Projetos

### [Finder](https://github.com/HugoGsilva/Finder)

Sistema web para monitoramento de guildas em servidores Rubinot/Tibia. O projeto envolve front-end, back-end, banco de dados, scraping, notificações e ambiente com Docker.

**Tecnologias:** TypeScript, React, Node.js, Express, PostgreSQL, Redis, Docker, Nginx.

### [Chatbot-Prolog](https://github.com/HugoGsilva/Chatbot-Prolog)

Chatbot sobre filmes da Netflix, usando lógica em Prolog no processamento das respostas. O projeto também conta com back-end em FastAPI, front-end em Angular e Redis para sessões.

**Tecnologias:** Prolog, Python, FastAPI, Angular, Redis, Docker.

### [Api-planilhas](https://github.com/HugoGsilva/Api-planilhas)

API para consultar CNPJs, processar arquivos XLSX e gerar planilhas em um modelo específico.

**Tecnologias:** Python, FastAPI, Docker, Portainer.

### [Atv-Final-Testes](https://github.com/HugoGsilva/Atv-Final-Testes)

Back-end de biblioteca com CRUDs e testes automatizados.

**Tecnologias:** Node.js, Express, Sequelize, Jest, Supertest, Playwright, Docker.

### [Atividade-2-dispositivos-moveis](https://github.com/HugoGsilva/Atividade-2-dispositivos-moveis)

Aplicativo de controle financeiro desenvolvido para estudos em desenvolvimento mobile.

**Tecnologias:** Dart, Flutter.
